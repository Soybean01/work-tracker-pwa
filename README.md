# 工作打卡PWA应用 - 快速安装指南

## 🚀 快速开始（3步安装）

### 第1步：启动本地服务器

打开项目文件夹，运行以下命令：

```bash
cd c:\Users\Soybean\Desktop\AMY\Obsidian\AMY\work-tracker-pwa
python -m http.server 8080
```

### 第2步：访问应用

浏览器打开：`http://localhost:8080`

### 第3步：安装到手机（可选）

**如果想在手机上使用，推荐使用Vercel免费部署：**

```bash
# 安装Vercel CLI
npm install -g vercel

# 部署
cd work-tracker-pwa
vercel

# 按提示登录，获得免费HTTPS链接
# 然后在手机浏览器访问链接，点击"添加到主屏幕"
```

## 📱 手机安装方法

### Android
1. Chrome浏览器访问应用
2. 菜单 → "添加到主屏幕"

### iPhone
1. Safari访问应用
2. 分享按钮 → "添加到主屏幕"

## 💡 使用提示

- **每日打卡：** 选择"上班"或"休息"→ 加班选项 → 保存
- **查看统计：** 自动显示本月工作天数、加班天数等
- **日历视图：** 可视化查看整月记录（蓝色=上班，橙色=加班）
- **导出数据：** 右上角下载按钮
