# 工作打卡 PWA 应用

一个简洁美观的工作打卡记录应用，使用PWA技术实现，可离线使用，支持安装到手机桌面。

## ✨ 功能特性

### 核心功能
- 📝 **每日打卡** - 快速记录上班/休息状态
- ⏰ **加班记录** - 记录加班时间（默认不加班，一键操作）
- 📊 **月度统计** - 自动统计工作天数、加班天数、加班时长
- 📅 **日历视图** - 直观查看整月打卡情况
- ✏️ **编辑历史** - 点击日历日期可修改历史记录
- 💾 **数据导出** - 支持导出JSON格式备份

### 高级特性
- 🌓 **主题切换** - 支持白天/黑夜模式，自动保存偏好
- 🗓️ **快速导航** - 年月选择器 + "回到今天"按钮
- 📈 **历史统计** - 查看任意月份的统计数据
- 🔒 **离线可用** - Service Worker缓存，完全离线使用
- 💻 **响应式设计** - 适配手机、平板、电脑

## 🚀 快速开始

### 在线访问
直接访问：**https://soybean01.github.io/work-tracker-pwa/**

### 安装到手机

#### Android (Chrome)
1. 访问网站
2. 点击浏览器菜单 ⋮
3. 选择"添加到主屏幕"
4. 点击"添加"

#### iOS (Safari)
1. 访问网站
2. 点击分享按钮 📤
3. 滚动找到"添加到主屏幕"
4. 点击"添加"

### 本地运行
```bash
# 1. 克隆仓库
git clone https://github.com/Soybean01/work-tracker-pwa.git
cd work-tracker-pwa

# 2. 启动本地服务器
python -m http.server 8080

# 3. 访问
浏览器打开 http://localhost:8080
```

## 📖 使用说明

### 打卡记录
1. 选择"上班"或"休息"
2. 如果上班，默认"不加班"（如需加班，选择"加班"并设置时间）
3. 点击"保存打卡记录"

### 查看统计
- 当月统计自动显示在顶部卡片
- 切换日历月份，统计会同步更新

### 编辑历史
- 点击日历上的任意日期
- 在弹窗中修改状态或删除记录

### 快速导航
- 使用 ← → 按钮切换月份
- 用年月选择器快速跳转
- 点击"今天"按钮回到当前月份

### 主题切换
- 点击右上角 ☀️/🌙 按钮
- 自动保存偏好设置

### 数据备份
- 点击右上角下载按钮导出JSON
- 建议每月备份一次

## 💡 设计理念

### 用户体验优先
- **少步操作** - 默认不加班，减少点击
- **视觉反馈** - 清晰的颜色区分（蓝色上班、橙色加班、绿色休息）
- **快速访问** - PWA安装后像原生APP一样

### 数据安全
- **本地存储** - 数据存在手机LocalStorage
- **隐私保护** - 不上传任何数据到服务器
- **定期备份** - 支持导出备份

### 美观现代
- **深色主题** - 护眼，省电（OLED屏幕）
- **浅色主题** - 白天使用更舒适
- **流畅动画** - 微动画提升体验

## 🔄 更新方式

### 自动更新
- Service Worker会自动检测新版本
- 关闭并重新打开应用即可更新

### 手动刷新
如果更新卡住了：
1. 浏览器设置 → 清除缓存（保留Cookie）
2. 刷新页面

## ⚠️ 注意事项

1. **数据不会丢失的情况：**
   - ✅ 清除浏览器缓存
   - ✅ 更新应用版本
   - ✅ 关闭浏览器/手机重启

2. **数据会丢失的情况：**
   - ❌ 清除浏览器数据/网站数据
   - ❌ 卸载浏览器
   - ❌ 手机恢复出厂设置

3. **建议：**
   - 每月1号导出一次数据备份
   - 重要数据可以发给自己做云备份
